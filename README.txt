== Description ==
 
Pryce uma plataforma perfeita para criação de campanhas promocionais, nela você poderá criar promoções e descontos exclusivos para cada tipo de cliente utilizando diversos parâmetros gerenciaveis.
Todo o gerenciamento é feito em uma plataforma dedicada e robusta para garantir o máximo de rapidez em sua loja virtual, veja abaixo alguns parâmetros de configuração que você poderá utilizar para gerar seus descontos:

- Datas especiais
- Intervalo de preço
- Categorias
- Marcas
- Afiliados
- Cep's
- Tipo de desconto (percentual ou nominal)

e mais...

Para começar a criar descontos para seus produtos você só precisa seguir estes 5 passos:

1. Faça o download deste plugin em sua loja e ative-o
2. Crie uma conta na plataforma pryce em https://pryce.app/users/signup/
3. Pegue seu token na sua [pagina de perfil do pryce](https://pryce.app/users/profile/)
4. Coloque seu token na sua loja virtual no menu: Woocommerce -> configurações -> "Token para pryce.app"
5. Crie seus descontos personalizados na pagina de descontos em https://pryce.app/benefits/ e pronto.

A partir de agora todos os preços de sua loja iram obedecer as regras de desconto que você criou no pryce!

== Installation ==
 
1. Faça o upload do plugin na sua instalacão e ative-o
2. Realize o seu cadastro em https://pryce.app/users/signup/
3. Obtenha seu token de acesso na sua [pagina de perfil](https://pryce.app/users/profile/)
4. Coloque o token nas configurações gerais do woocommerce na opcão `Token para pryce.app`

Crie suas campanhas de desconto na nossa plataforma e aproveite!
