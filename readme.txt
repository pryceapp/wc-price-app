== Description ==
 
Pryce uma plataforma completa para criação de campanhas promocionais segmentadas, com gerenciamento e API's simples você poderá criar promoções e descontos exclusivos para cada tipo de cliente utilizando diversos parâmetros gerenciaveis, todo o gerenciamento é feito em uma plataforma dedicada e robusta para garantir o máximo de rapidez em sua loja virtual, veja abaixo alguns parâmetros de configuração que você poderá utilizar para gerar seus descontos:

- Datas especiais
- Intervalo de preço
- Categorias
- Marcas
- Afiliados
- Cep's
- Tipo de desconto (percentual ou nominal)

e mais...

== Installation ==
 
1. Faça o upload do plugin na sua instalacão e ative-o
2. Realize o seu cadastro em https://pryce.app/users/signup/
3. Obtenha seu token de acesso na sua [pagina de perfil](https://pryce.app/users/profile/)
4. Coloque o token nas configurações gerais do woocommerce na opcão `Token para pryce.app`

Crie suas campanhas de desconto na nossa plataforma e aproveite!
